# Node.js Express API Skeleton

Une base solide et légère pour construire des API REST avec **Node.js** et **Express.js**. Ce projet inclut déjà une gestion d'erreurs robuste et des middlewares essentiels.

## Fonctionnalités incluses

* **Serveur Express** : Configuré et prêt à l'emploi sur le port 3000.
* **Middlewares de Sécurité** : Validation automatique du `Content-Type: application/json` pour les requêtes entrantes.
* **Gestion des Erreurs 404** : Capture automatique des routes inexistantes.
* **Error Handler Global** : Centralisation de la gestion des erreurs pour des réponses JSON cohérentes.
* **Prêt pour le CRUD** : Structure optimisée pour ajouter vos propres ressources (utilisateurs, tâches, etc.).
